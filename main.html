Puzlet

Scientific computing in the browser

Puzlet uses a special brew of CoffeeScript tailored for scientific computing. If you're familiar with the likes of Matlab, Octave, or Python/SciPy, you'll feel right at home. The heart of Puzlet is the blab — short for web lab, like blog is short for web log. Blabs are grouped, by topic, within GitHub organizations:
  • Science, Technology, Enginering & Mathematics (stemblab)
  • Climate Science Web Laboratory (climateblab)
